# TemanKasir - Simple Free Webview-Based Software For Cashier Solution
TemanKasir ialah project software berbasis webview yang dikembangkan menggunakan framework Next.JS dan React dengan bantuan Tauri untuk membantu keperluan kasir

<br>

# Fitur
### Auto-Recipe
Pada TemanKasir, saat anda menekan tombol "Checkout", akan ter-generate struk otomatis dengan identitas toko dan detail pembelian

### Data produk yang tersimpan rapih
TemanKasir memiliki sistem untuk menyimpan data-data produk dengan rapih dan terkelola secara lokal. Pengguna juga dapat mengubah data produk melalui fitur yang tersedia

### Cutomizable identitas toko
TemanKasir memiliki opsi untuk mengubah dan menambahkan nama, alamat dan nomor telephone dari toko dan dapat muncul pada bagian header/atas struk

<br>

# Fitur yang akan mendatang
***Catatan:** Fitur mendatang ini tidak dapat di prediksi kapan akan rilisnya dikarenakan kesibukan developer.*
*Jika ada yang berminat untuk menambahkannya, bisa langsung Pull Request aja, ya!*

### Stok Pada Produk
Setiap pengguna melakukan checkout/transaksi, otomatis barang berkurang sesuai barang yang ada dalam keranjang.

### Riwayat transaksi/checkout
Setiap transaksi dibuat (saat user klik checkot button), riwayat transaksi akan dibuat dan disimpan.
Pengguna bisa melihat riwayat itu sewaktu-waktu melalui dashboard.

### Rekapitulasi
Setiap checkout/transaksi yang tersimpan dapat di cek dan dihitung berapa laba kotor, laba bersih, rugi, dsb. melalui dashboard.

### Dashboard
Menampung segala keperluan administrasi, termasuk didalamnya rekapitulasi dan riwayat transaksi.

### Input opsi pembayaran (Tunai atau Hutang)
Untuk sekarang, TemanKasir hanya dapat membantu mencetak struk. Kedepannya, akan ditambahkan juga opsi pembayaran, apakah tunai atau masuk kasbon/hutang.

# Menambahkan Produk
Untuk dapat menambahkan produk, pengguna dapat mengikuti langkah berikut:

1. Dari halaman awal, klik menu (tengah bawah, tombol ke 3)
2. Pilih **"List Produk"**
3. Anda akan diarahkan pada halaman **Managemen Produk**, setelahnya pilih opsi **Tambah Produk** pada pojok kanan bawah
4. Isi form sesuai dengan data
5. Setelahnya, klik **Simpan**
6. Produk berhasil ditambahkan!
